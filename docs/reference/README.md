# docs/reference — 独立キットの要件・設計文書（出自・凍結）

`seminar-kit` / `mfg-kit` は、この汎用キット（aiseed-migration-kit）が
生まれる前に、業務ごとに個別実装していた**前身プロジェクト**。両者は
`sites/seminar` / `sites/mfg` に「定義ファイルの差し替えだけ」で再現され
（DESIGN §15・`tests/test_showcase.py`）、独立リポジトリは 2026-07-18 に
アーカイブ・削除した。

ここに残すのは、その**要件・設計文書だけ**（業務知識は再生成できないため）。
実装コード（backend/app/site/office）は汎用エンジン `src/amig/` に置き換わった
ので持ち込んでいない。

- `seminar-kit/` … 研修・セミナー管理の要件一式（申込→名簿→台帳・一斉連絡・
  M365 代替(07)・官公庁向け無害化(08)・AI の考え方(05)・軽量版(06)）
- `mfg-kit/` … 製造業の見積・受注の要件一式（見積依頼→受注・Office連携・
  QR・基幹連携(02)・E2E(05)）

これらは**凍結された背景資料**。現行の正は各サイトの `site.yaml`＋様式
プロファイルと、キット本体の `DESIGN.md`。
