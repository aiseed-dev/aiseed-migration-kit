= お問い合わせ

ご相談・お問い合わせ全般の様式です。記入のうえメール本文として
お送りください。

ご所属(企業・団体名):: [varchar(100), not null]
  例: 株式会社○○製作所
お名前:: [varchar(50), not null]
  例: 山田 太郎
電話番号:: [varchar(20), not null]
  例: 088-000-0000
メールアドレス:: [varchar(255), not null, check (メールアドレス ~ '^[^@]+@[^@]+$')]
  例: taro@example.co.jp
ご用件:: [text, not null]
  ○○の試験についてご相談したい、など自由にお書きください。
