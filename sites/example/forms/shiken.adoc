= 依頼試験申込

依頼試験の申込様式です。試験項目・数量をご確認のうえお申し込みください。

企業・団体名:: [varchar(100), not null]
  例: 株式会社○○製作所
ご担当者名:: [varchar(50), not null]
  例: 山田 太郎
電話番号:: [varchar(20), not null]
  例: 088-000-0000
メールアドレス:: [varchar(255), not null, check (メールアドレス ~ '^[^@]+@[^@]+$')]
  例: taro@example.co.jp
試験項目:: [varchar(50), not null, check (試験項目 in ('引張試験', '硬さ試験', '成分分析'))]
希望日:: [date]
  例: 2026-08-01(和暦でも構いません)
数量:: [integer, not null, check (数量 between 1 and 100)]
  本数・個数を半角数字で。
備考:: [text]
