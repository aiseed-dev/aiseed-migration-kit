= 見積依頼

製品カタログの製品番号を明細にご記入ください。図面・仕様書がある場合は
このメールに添付してください。

会社名:: [varchar(100), not null]
  例: 株式会社○○製作所
ご担当者名:: [varchar(50), not null]
メールアドレス:: [varchar(255), not null, check (メールアドレス ~ '^[^@]+@[^@]+$')]
  見積書の送付先になります。
電話番号:: [varchar(20), not null]
明細:: [text, not null]
  1行に1件、「製品番号 数量 個別仕様(任意)」の順にお書きください。
  例: P-1024 2 黒色・座面高さ45cm
希望納期:: [date]
  例: 2026-09-30(和暦でも構いません)
備考:: [text]
  用途・予算感・納品先など、補足があればお書きください。
