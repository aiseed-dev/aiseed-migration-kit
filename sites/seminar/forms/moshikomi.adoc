= 受講申込

受講者が1名または2名の場合、使わない組は空欄のままにしてください。
フリガナは全角カタカナで(ひらがなで書かれた場合はカタカナに直して
受け付けます)。

企業・団体名:: [varchar(100), not null]
  例: 株式会社○○製作所
企業・団体名フリガナ:: [varchar(100), not null, check (企業・団体名フリガナ ~ '^[ァ-ヶー ]+$')]
  例: カブシキガイシャマルマルセイサクショ
ご担当者名:: [varchar(50), not null]
ご担当者名フリガナ:: [varchar(50), not null, check (ご担当者名フリガナ ~ '^[ァ-ヶー ]+$')]
メールアドレス:: [varchar(255), not null, check (メールアドレス ~ '^[^@]+@[^@]+$')]
  受領メール(申込番号つき)の送付先になります。
郵便番号:: [varchar(8), not null]
  例: 770-0000
住所:: [varchar(200), not null]
電話番号:: [varchar(20), not null]
FAX:: [varchar(20)]

受講者1氏名:: [varchar(50), not null]
受講者1フリガナ:: [varchar(50), not null, check (受講者1フリガナ ~ '^[ァ-ヶー ]+$')]
受講者1所属・役職:: [varchar(100), not null]
受講者1メールアドレス:: [varchar(255), not null, check (受講者1メールアドレス ~ '^[^@]+@[^@]+$')]
  ZOOM 接続情報の送付先になります。
受講者1参加場所:: [varchar(10), not null, check (受講者1参加場所 in ('会場', 'ZOOM', 'サテライト'))]

受講者2氏名:: [varchar(50)]
受講者2フリガナ:: [varchar(50), check (受講者2フリガナ ~ '^[ァ-ヶー ]+$')]
受講者2所属・役職:: [varchar(100)]
受講者2メールアドレス:: [varchar(255), check (受講者2メールアドレス ~ '^[^@]+@[^@]+$')]
受講者2参加場所:: [varchar(10), check (受講者2参加場所 in ('会場', 'ZOOM', 'サテライト'))]

受講者3氏名:: [varchar(50)]
受講者3フリガナ:: [varchar(50), check (受講者3フリガナ ~ '^[ァ-ヶー ]+$')]
受講者3所属・役職:: [varchar(100)]
受講者3メールアドレス:: [varchar(255), check (受講者3メールアドレス ~ '^[^@]+@[^@]+$')]
受講者3参加場所:: [varchar(10), check (受講者3参加場所 in ('会場', 'ZOOM', 'サテライト'))]
